# Noah Koeng Portfolio

> **ALWAYS run `git pull` before doing anything.**

## Project Structure

| Location | Contents |
|---|---|
| `public/` | All images |
| `src/pages/` | All page code |
| `src/constants/` | Colors (`colors.ts`) |

## Running Locally

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploying Changes to Production

1. Stop the local server with **Ctrl + C**
2. Save all files: **File → Save All** (top-left menu)
3. Run the following commands in the terminal in order:

```bash
git add .
git commit -m "[describe your changes here]"
git push
```
